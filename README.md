## 注意事项

- 开始 `Commit` 之前请先阅读 <a rel="readme" href="地图配置提交规范.md">地图配置提交规范</a>
- 禁止一个PR修改多个地图配置及翻译文件
- 文本换行符统一为LF
- 文本保存格式统一保存为UTF8-无BOM
- 文本缩进为统一4空格缩进, JSON/JSONC为2空格缩进
- 文件内容禁止使用中文标点
